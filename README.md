# 🏏 IPL 2026 SQL Analysis

## Project Overview
Analysis of IPL 2026 ball-by-ball data using MySQL. 
Dataset contains 1382+ deliveries from IPL 2026 season.

## Dataset
- Source: Kaggle (IPL 2026 ball by ball dataset by Sahil Tailor)
- Tables: matches, teams, players, deliveries

## Analysis Performed
- Top run scorers of IPL 2026
- Top wicket takers
- Most expensive bowlers (runs given)
- Most wides bowled
- Toss vs match win correlation
- Player of the Match leaders
- Team-wise player roster using JOINs
- Venue-wise match count

## SQL Concepts Used
- DDL: CREATE TABLE, DROP TABLE
- DML: INSERT, UPDATE, DELETE
- Joins: INNER JOIN
